# Documentación Técnica - CRM App

## Información General

**Nombre de la aplicación:** CRM con clientes y notas
**Stack tecnológico:** HTML + JavaScript
**Repositorio:** https://github.com/luisernestosalas/app-rx9xa
**URL de despliegue:** https://app-rx9xa-fxcx58u0r-luisernestosalas-2775s-projects.vercel.app

## Descripción del Proyecto

Sistema CRM (Customer Relationship Management) básico desarrollado con tecnologías web frontend que permite la gestión de clientes y sus respectivas notas asociadas.

## Arquitectura Técnica

### Frontend
- **HTML:** Estructura y marcado de la interfaz de usuario
- **JavaScript:** Lógica de negocio y manipulación del DOM
- **Tecnología:** Aplicación web del lado del cliente (Client-side)

### Despliegue
- **Plataforma:** Vercel
- **Tipo:** Despliegue estático/frontend

## Funcionalidades Principales

### Gestión de Clientes
- Creación de registros de clientes
- Visualización de listado de clientes
- Edición de información de clientes
- Eliminación de registros de clientes

### Sistema de Notas
- Asociación de notas a clientes específicos
- Creación de nuevas notas
- Visualización de notas por cliente
- Gestión y actualización de notas existentes
