## ¿Qué es esta aplicación?

Esta es una aplicación CRM (Customer Relationship Management) sencilla. Te permite:
- Guardar información de tus clientes
- Agregar notas sobre cada cliente
- Organizar tu base de datos de clientes

La aplicación funciona en tu navegador web. No necesitas instalar nada.
