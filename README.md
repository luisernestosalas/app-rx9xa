# CRM App - Sistema de Gestión de Clientes y Notas

Una aplicación web simple y eficiente para gestionar clientes y sus notas asociadas, desarrollada con tecnologías web modernas.

## 🚀 Demo en Vivo

**[Ver Demo](https://app-rx9xa-fxcx58u0r-luisernestosalas-2775s-projects.vercel.app)**

## 📋 Características

- ✅ **Gestión de Clientes**: Agregar, editar y eliminar información de clientes
- ✅ **Sistema de Notas**: Crear y gestionar notas asociadas a cada cliente
- ✅ **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- ✅ **Responsivo**: Funciona en dispositivos móviles y de escritorio
- ✅ **Almacenamiento Local**: Los datos se guardan en el navegador
- ✅ **Sin Backend**: Aplicación completamente del lado del cliente
